I want to create a simple calendar system for World Cup 2026 matches.

I make reference to the `sample/calendar system.md` about how the calendar system works. In this stage I would like to create mainly two things: an ics file of the calendar and the supportive json database.

The data source is mainly FIFA website.

The calendar title is 'World Cup 2026'.

An event is in the format:

BEGIN:VEVENT
UID:fifa-wc26-match-73@worldcup-calendar
DTSTAMP:20260629T114138Z
DTSTART;TZID=America/Los_Angeles:20260628T120000
DTEND;TZID=America/Los_Angeles:20260628T140000
SUMMARY:R32 - South Africa vs Canada (M73)
DESCRIPTION:FIFA World Cup 2026 Match 73\nRound of 32 - South Africa vs Canada\n\nScores and fixtures: https://www.fifa.com/en/tournaments/mens/worldcup/canadamexicousa2026/scores-fixtures \n\n Match page: https://www.fifa.com/en/match-centre/match/17/285023/289287/400021518
LOCATION:Los Angeles Stadium (Los Angeles)
URL:https://www.fifa.com/en/match-centre/match/17/285023/289287/400021518
STATUS:CONFIRMED
END:VEVENT

Explanation on the title: The match category is either: GROUP, R32, R16, QF, SF, FINAL, 3RD

For R16 and beyond, the match opponents are yet to be determined. In this case, the opponents are marked as W74, W75, etc. For example, 'R16 - Canada vs W75 (M90)'.
